Windows 11 实体机安装 YES OR NO？
作者：电脑爱好者
上期文章我们介绍了因为 TPM 2.0 而无法安装 Windows 11 的事情，许多大佬狂写软件来解决问题。其实，只要用到一个良心系统(或者叫预安装环境)就可以，它就是“微PE”。

微PE下载网址：http://www.wepe.com.cn/download.html?tdsourcetag=s_pcqq_aiomsg

至于白嫖方法，非常简单：选择2.0或2.1版，点击“我已捐赠，继续查看下载地址”就可以。当然如果想大力支持一下作者也可以。

Windows 11 下载地址：ed2k://|file|22000.51.210617-2050.CO_RELEASE_SVC_PROD2_CLIENTMULTICOMBINED_UUP_X64FRE_ZH-CN.ISO|5727295488|F27604BE2EF9E4D3A20886157A8297D3|/ (请使用迅雷下载)

下载并打开后，选择右下角那个U盘图标，插入U盘，大部分都是默认，格式化的文件格式建议选NTFS(当然exFAT和FAT32也是没有问题的)。

请注意：安装前请注意备份数据！

装好后，在那个U盘原名的分区里复制 Windows 11 的镜像，然后就可以亮机啦！进入你电脑的BIOS(不同电脑按键可能不同，请自行百度)，选择U盘，打开“Windows 安装器”，第一大栏选中你U盘里的镜像，第二大栏选中你的 EFI / ESP 分区，若找不到请用“分区工具 DiskGenius”挂载，第三大栏中选择系统盘，点击安装，确定，等待它自己的安装。

感觉新的OOBE有苹果内味了……

你学会安装 Windows 11 了吗？
